1.0 - Buttons

Use any of the available button classes to quickly create a styled button.

Default styling

Highlight the button when hovered.

:hover

when activated

:active

when disabled

.disabled

Larger size

.btn-lg

smaller size

.btn-sm

extra smaller size

.btn-xs

buttons into blocks

.btn-block

 
<button type="button" class="btn btn-default {$modifiers}">Default</button>
<button type="button" class="btn btn-primary {$modifiers}">Primary</button>
<button type="button" class="btn btn-success {$modifiers}">Success</button>
<button type="button" class="btn btn-info {$modifiers}">Info</button>
<button type="button" class="btn btn-warning {$modifiers}">Warning</button>
<button type="button" class="btn btn-danger {$modifiers}">Danger</button>
<button type="button" class="btn btn-link {$modifiers}">Link</button>
